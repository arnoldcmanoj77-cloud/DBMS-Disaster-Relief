# Disaster Relief Management System

## 📌 Project Description
A Java-based GUI application developed using Swing and MySQL for managing disaster relief resources.

## 💻 Technologies Used
- Java (Swing GUI)
- MySQL
- JDBC (MySQL Connector)
- Git & GitHub

## 🔧 Features
- Add disaster resources
- Store data in MySQL database
- GUI-based interaction
- Database connectivity using JDBC

## 🗄 Database
Database Name: disaster_db  
Table: resources
